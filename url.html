What is a URL

URL = Uniform Resource Locator.

URL is the address of a website.

A URL points to a unique destination.

Absolute URL

An absolute url is a url containing protocal("http://" or "https://"), "www" signal, and domain name. We can access an absolute url from public.

Relative URL

On the other hand, a relative url is an url that does not specify the protocal, "www" signal, or domain name. We cannot access a relative url from public. It can only be accessed within a webpage under the same domain name.
